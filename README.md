# Smart Agile Sprint Management Tool with Analytics

A web-based Agile project management application developed using **HTML, CSS, JavaScript, LocalStorage, and Chart.js**. The system helps teams manage sprints, tasks, Kanban workflows, and analytics dashboards.

---

## 📌 Project Overview

The Smart Agile Sprint Management Tool enables users to:

- Manage team members and team leader
- Create and manage multiple sprints
- Assign tasks to team members
- Track task progress using a Kanban board
- Monitor team performance through analytics
- Visualize sprint progress using charts
